The value of snap confinement

This report contains both Actionable (USN available) and Confined Mitigation (no USN or official patch available upstream) findings.

The CI workflow publishes OSV reports for visibility and fails only when the scanner itself errors. Known-vulnerability exit code 1 is treated as a warning. Unlike conventional deployments, a strictly confined snap executes within a sandbox: process capabilities and host interactions are restricted by AppArmor profiles, seccomp filters, and a read-only SquashFS filesystem.
